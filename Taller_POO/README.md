# Taller de Programación Orientada a Objetos (POO)

Este proyecto contiene un ejemplo de aplicación de los principios básicos de la Programación Orientada a Objetos en Java.

## Estructura del Proyecto

```
Taller_POO/
│
├── src/
│   ├── Producto.java
│   ├── Cliente.java
│   ├── Impresion.java
│   ├── Foto.java
│   ├── Camara.java
│   ├── Pedido.java
│   └── Main.java
│
└── README.md
```

## Cómo ejecutar

1. Abre el proyecto en tu IDE (Eclipse, NetBeans o VSCode).
2. Compila y ejecuta `Main.java`.
3. Se mostrará información del pedido y la impresión de fotos.
